# Java Swing Calculator

Calculadora desktop desenvolvida em Java Swing para praticar:

- Programação Orientada a Objetos
- Java Swing
- Eventos (ActionListener)
- Tratamento de exceções

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Limpar visor
